# 电机理解
- 这个电机内置了处理器，不用直接控制其电流，可以直接给其发送速度、位置、转矩指令，其内部处理器会进行处理达到目标，从而减轻上位机计算压力。
# T_ff转矩给定值参数
- 映射关系
  - ![img.png](img.png)
  - T_ff为直接给定的转矩（力），是开环控制，位置控制和速度控制都是通过转矩来控制的

# CAN控制
- MIT控制
- 操作流程
  - 1、使能
  - 2、数据
    - MIT模式下由于需使用浮点数进行控制，但二进制传输不方便传输浮点数，所以在上位机和电机之间需要一个转码，在传输过程中使用整数，在上位机和电机上使用浮点数。
  - 3、失能
